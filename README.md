# GEO Satellite Debris Observation Scheduling

## Overview
Built during ISRO internship to optimize GEO debris observation using a ground-based telescope under real-world constraints.

## Features
- Greedy (EDF) scheduler (~166 observations/night)
- Genetic Algorithm optimization (quality improvement)
- Multi-night scheduling framework

## Tech Stack
Python, Optimization Algorithms, Genetic Algorithms

## Results
- Near physical limit scheduling achieved
- Trade-off between quantity vs quality demonstrated
